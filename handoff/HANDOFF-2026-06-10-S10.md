## Validated Prompt Rules (new this session — already written to vault)

1. **Scene material pull palette** — warm materials (brick, timber, jalousie) + warm lighting anchor (Bradford Young, pendant industrial) → amber/yellow drift. Fix: `cool midtones desaturated, cool-dominant zero amber push` in `color_grade`.
2. **Atmosphere token controls pose energy** — `calm professional focus` → model defaults to static pose. `lively bustling crowd energy` / `electric forward momentum` → dynamic pose.
3. **No redundancy within a field** — one strong token, no reinforcement. Model reads the token; trust it.
4. **Feminine gesture tokens validated** — `feminine hip displacement` in `pose`, `feminine wrist tilt` in `gesture`. One token each.
5. **No narrative labels in any field** — `expression` and `gesture` must be physical description only. No `making a point, controlled`, no `person mid-sentence making a case`.
6. **Live actor scenes → populated environment** — communal/co-working space with background activity, not empty home office.
7. **`brows lifted` → model renders worry/shock, not positive surprise** — for positive emotion, use Duchenne anchor: `cheeks lifted Duchenne pull, mouth open mid-exhale of delight, eyes crinkling outer corners`.
8. **Never direct face toward background figure** — character's first reaction is to device screen. Face toward phone. Body can turn.
9. **Off-axis angle pattern** — `camera left-of-subject looking right across foreground object` + `subject right-third frame` + foreground object fills lower-left quadrant. Validated S2.
10. **Color grade for domestic warm scenes** — `cool midtones desaturated, cool-dominant zero amber push` mandatory when scene has warm materials.
