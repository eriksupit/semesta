## Validated Prompt Standard (Session 5 must enforce)

Fully documented in `prompt-rules-image-generation.md`. Key points:

1. **Format:** JSON with explicit fields — `scene`, `action`, `camera`, `lighting`, `style`, `color_grade`, `aspect_ratio`
2. **Style:** Photorealistic cinema still — ARRI Alexa, Kodak 2383, anamorphic, prestige drama register
3. **Photography tokens:** Always include lens name, film stock, DP reference, color grade by name
4. **Action:** Physical concrete event per prompt — no mood descriptions
5. **Constraint:** Positive statements only — no negative prompts
6. **Aspect ratio:** 16:9, mandatory on every prompt
7. **Scene logic:** The space must be dominated by something — not an object placed in empty space
8. **S0 tone:** Normal, bright, clinical — unsettling only from the empty chair. NOT dark, NOT surveillance.
