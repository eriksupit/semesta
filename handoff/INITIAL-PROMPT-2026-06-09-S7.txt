== STANDAR PROMPT BAKU (dari S2 — wajib) ==
Urutan field: mood → color_grade → style → character_identity → character_style → scene → subject_state → action → [on_screen_graphic + graphic_proximity] → framing → pose → gesture → expression → angle → camera → lighting → aspect_ratio.

Aturan kritis (detail penuh di prompt-rules-image-generation.md):
- Token warna HANYA di color_grade + mood. Zero warna di facial_features/wardrobe/scene/lighting.
- facial_features = kualitas/tekstur (healthy glow, luminous complexion), zero nama warna.
- wardrobe = garment construction tokens (fabric + construction + fit + colorway desaturated). BANNED: batik/floral/ethnic print, terracotta/ochre/amber/golden.
- scene = design interior tokens (material finish + fixture + surface). BANNED: label geografis/etnis.
- color grade satu sistem Roger Deakins; hanya midtones bergeser (cool desaturated / slightly warmer / neutral cool).
- NEGASI DILARANG semua field (no X, not X, un-, non-). Pakai token positif.
- age = satu angka eksplisit, +5 tahun aging bias.
- ethnicity = etnis spesifik (modern Javanese features) atau urban (modern Jakartan features). BANNED: Indonesian features, Betawi features.
- beauty dikalibrasi kelas: jelita/tampan/gagah (middle ke atas), cantik (lower-middle/domestic).

Teknik 2 floating UI graphic (POLA BAKU S2) — dua field berpasangan WAJIB:
- on_screen_graphic = "air-composited superimpose over scene — frosted glass card panel rounded corners semi-transparent white fill subtle inner glow edge soft gaussian blur backdrop — element 1: '[teks]' clean sans-serif large weight teal-cyan accent color centered on card — element 2: '[badge]' pill-shaped badge teal-cyan solid fill white text medium weight centered below element 1 — card panel soft drop shadow layered depth over scene."
