- **oMLX vs OpenRouter:** oMLX (local inference) requires model stored locally (~14GB) + min 16GB unified memory. OpenRouter chosen for M2/M3 MacBook compatibility regardless of memory spec. Evidence: oMLX GitHub docs + session decision.
